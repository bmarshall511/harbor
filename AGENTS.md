# Agent Operating Guide

This repository is intended to be worked on by Claude Code, Cursor, and human developers.

## Required Behavior
- Read `CLAUDE.md` and relevant rules before making meaningful changes.
- Inspect the existing code before proposing architecture.
- Ask direct clarifying questions when requirements are ambiguous.
- Prefer small, clean abstractions and minimal churn.
- Keep implementation production-ready, typed, tested, and documented.
- Preserve accessibility, keyboard support, permissions, auditability, and open-source contributor friendliness.
